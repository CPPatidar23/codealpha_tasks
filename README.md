# codealpha_tasks
Student Grade Management System

A simple and efficient Java console-based application for managing student grades. The application enables users to input student information, calculate grade statistics, and generate a summary report. It is built using Object-Oriented Programming (OOP) concepts and Java Collections (ArrayList) for dynamic data storage.

Features
Add student names and grades
Store records using ArrayList
Calculate average score
Find highest and lowest scores
Display a summary report of all students
Simple and user-friendly console interface
Easy to extend with additional features
Technologies Used
Java
Object-Oriented Programming (OOP)
ArrayList
Scanner Class
Java Collections Framework

How to Run

Clone the repository:

git clone https://github.com/your-username/Student-Grade-Management-System.git
Open the project in your preferred Java IDE (VS Code, IntelliJ IDEA, Eclipse, or NetBeans).

Compile the source files:

javac Student.java StudentGradeManager.java

Run the application:

java StudentGradeManager
